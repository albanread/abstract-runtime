## Usage in Code

To use PETSCII characters in the abstract runtime, you can:

1. **Direct character codes**: Use the hex values directly as character codes
   ```cpp
   print_at(x, y, "\x70");  // Box drawing: down and right corner
   ```

2. **Screen codes**: Convert screen codes to character values
   ```cpp
   char box_corner = 0x70;  // Screen code for top-left corner
   ```

3. **Unicode mapping**: If the font system supports Unicode, use the Unicode values
   ```cpp
   print_at(x, y, "\u250C");  // Unicode box drawing character
   ```
